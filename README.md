# Reference Docs

- [Head servo](https://https://www.amazon.com/dp/B07RFRLRV8), stronger servo needed for R2D2 head rotation (only 1 needed)
- [Arduino IDE](https://www.arduino.cc/en/Main/Software), R2Duino was tested with version 1.8.19
